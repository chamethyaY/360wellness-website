# 360 Wellness Website

A responsive **health and wellness website** designed to provide users with information, resources, and interactive features related to healthy living, nutrition, volunteering, and personal wellbeing.

## Project Overview
The **360 Wellness Website** was developed as an academic web development project to create a user-friendly and visually appealing platform focused on health and wellness awareness.

The website includes multiple interactive pages such as:
- Homepage
- Content pages
- User profile
- Volunteer page
- Feedback form
- Team page
- Sitemap
- Splash screen
- Validation pages

The main objective of this project was to design a complete multi-page website with a clean UI, consistent navigation, and responsive layout.
